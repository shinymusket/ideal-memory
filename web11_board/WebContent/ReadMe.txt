모델2 기반 게시판

1. 데이터베이스 구축(테이블 생성, 시퀀스 생성)
2. 샘플 데이터 삽입과 확인(commit)

3. DBCP 셋팅
4. 각종 라이브러리를 복사(OJDBC8, JSTL)

5. VO 객체 제작
6. DBManager 객체 제작(DB 연결과 해제)
7. DAO 객체를 제작
 - 싱글턴 패턴
 - 각 쿼리를 호출할 메서드
 	- 모든 게시물 조회
 	- 게시물 등록
 	- 게시물 조회시 카운트 증가
 	- 게시물 상세보기
 	- 게시물 수정
 	- 게시물 삭제
 	- 게시물 암호 확인
8. CSS
9. 요청을 처리하기 위한 서블릿(BoardServlet)
10. 요청을 기능으로 연결시켜줄 클래스(ActionFactory)
11. 각 기능을 가진 클래스를 묶어줄 인터페이스(Action)

12. 모든 게시물 목록 보기 기능 
 - 게시물 보기 기능을 처리할 클래스(Action 구현 받은 클래스)
	- ActionFactory 등록
 - 화면을 보여주기 위한 뷰

 ------------------------------------------
 13. 글 등록
  1) 폼 요청
  	- 게시물 등록 폼을 보여주는 기능을 처리할 클래스(Action 구현 받은 클래스)
  	- 게시물 등록 폼을 보여주기 위한 뷰
  2) 게시물 등록 요청
  	- 게시물을 등록하기 위한 기능을 처리할 클래스(Action 구현 받은 클래스)
  	- DB에 등록하기 위한 DAO 메서드
  	- 결과를 반환하기 위한 뷰(목록 보기)
  - 각 기능 클래스를 ActionFactory에 등록
  
 -----------------------------------------
 14. 글 상세 보기
  - 게시물 상세 보기를 위한 기능을 처리할 클래스(Action 구현 받은 클래스)
  - DB에서 해당 게시글 정보를 알아오기 위한 DAO메서드
   - 조회수 증가 DAO메서드
  - 해당 정보를 반영한 뷰
  - 기능 클래스를 ActionFactory에 등록
 
 -----------------------------------------
 15. 수정/삭제를 체크하기 위한 암호입력
  - 암호 체크 폼으로 이동하기 위한 기능을 처리할 클래스(Action 구현 받은 클래스)
  - 암호 체크 폼을 보여주기 위한 뷰 => javascript를 통한 새창 열기
  
  - 암호 체크 기능을 처리하기 위한 클래스
  - DB에서 암호를 확인하기 위한 DAO메서드
  - 암호 체크 실패시 -> 암호체크 폼으로 돌아가기
  - 암호 체크 성공시  -> 체크 성공 뷰
  					-> 글 수정
  					-> 글 삭제
  					
 -----------------------------------------
 16. 글 수정
   1) 
     - 글 수정 폼으로 이동하기 위한 클래스
     - 글 수정 폼을 보여주기 위한 뷰
    
   2)
     - 글 수정 기능을 위한 클래스
     - DB에 글 수정을 처리하기 위한 DAO 메서드
     - 글 수정후 이동할 페이지 목록보기로 이동 
  - ActionFactory에 클래스 등록
 -----------------------------------------
 17. 글 삭제
  - 글 삭제 기능을 처리하기 위한 클래스
  - DAO 글 삭제 기능을 하기 위한 메서드
  - 글 삭제 후에는 목록보기로 이동
  - ActionFactory에 클래스 등록
